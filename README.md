#### 🖇 Dependências
Instalar as dependências do projeto:
```
npm install
```

#### ▶️ Ambiente
Rodar o ambiente de desenvolvimento:
```
npm run serve
```

#### 🛠 API fake
Instalar o json-server globalmente:

```
npm install -g json-server
```

Iniciar a api fake de leilões;

```
json-server --watch db.json
```
